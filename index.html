    Создайте страницу с круглым элементом, при клике на который цвет меняется в следующем порядке: зеленый - желтый - красный - зеленый - желтый - красный и т.д. Один клик — одна смена цвета. Изначально цвет элемента чёрный.

  • Задача 5 — Перехват формы

    Создайте страницу с формой, где есть поле ввода, кнопка отправки и текстовый блок с id duplicateField. При вводе текста в поле ввода (input) текст должен дублироваться в текстовый блок с id duplicateField. При нажатии кнопки «‎Вывести текст в консоль и удалить из формы»‎ текст, написанный в input, должен быть выведен в консоли, а поле ввода и duplicateField должны быть очищены.
